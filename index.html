Fuzzy Logic-Based Tumor Immune Escape Modeling

This project explores the use of fuzzy logic to model VISTA-mediated tumor immune escape mechanisms. The goal is to predict immune evasion thresholds and help develop targeted cancer immunotherapies.

Key skills used:

  • Fuzzy logic modeling
  • Mathematical modeling & simulation
  • Data analysis
  • Machine Learning
  • NumPy
  • scikit-learn
  • Matplotlib
  • Technical writing
